Flip-Flop-Erweiterungskarte
Veröffentlichung

1.14.4

Auf Flip-Flop-Erweiterungskarten werden Daten gespeichert, die vom Flip-Flop-Hub ausgelesen werden.
In der Workbench können Erweiterungskarten mit Pins, Kabeln, Speicherkarten und Widerständen zu den Flip-Flop-Erweiterungskarten erweitert werden.

Herstellung

Zur Herstellung öffnet der Spieler mit Rechts-Klick die Workbench. Dort kann ausgewählt werden, welche Gatter Logik auf der Erweiterungskarte gespeichert werden. In der GUI der Workbench wird angezeigt, welche Materialien eingelegt werden müssen. Nach dem einlgen kann der Spieler die Logik-Gatter-Erweiterungskarte herausnehmen. Verwendung

Der Spieler kann durch einen Rechts-Klick auf den Flip-Flop-Hub das GUI öffnen. Dort kann auf der rechten Seite eine Flip-Flop-Erweiterungskarte eingelegt werden. Je nach Einlage ändert sich die Rechnerlogik im Block und die Textur des PC auf dem Block. Eine Übersicht über die Formeln der Logik gibt es hier.
